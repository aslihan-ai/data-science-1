# data-science-1

LEVEL 1: Reverse a String
 Description

This program takes a string input from the user and returns the reversed version of it using Python slicing.

 Key Concept
String slicing ([::-1])
 Example
Enter a string: hello
Reversed string: olleh

 LEVEL 2: Count Vowels in a String
 Description

This program counts how many vowels (a, e, i, o, u) are present in a given word or sentence.

 Key Concepts
Loops (for)
Conditional statements (if)
String membership (in)
 Example
Enter a word or sentence: ChatGPT
Number of vowels: 1
